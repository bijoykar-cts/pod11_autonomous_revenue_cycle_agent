Hackathon MVP

ICD-10 Coding Pipeline

A local clinical coding assistant that extracts evidence, recommends ICD-10-CM/PCS candidates, validates against a configured corpus, and keeps human review visible.

Safety > speed > polish

Problem

Coding work is both interpretive and brittle.

Clinical notes contain negation, history, specificity gaps, and procedure details. A valid code still needs evidence and a review path, not just a matching description.

Pipeline

Three deterministic agents, one auditable result.

1Note input
2Evidence extraction
3Code recommendation
4Corpus validation
5Human review state

Safety Contract

The MVP shows uncertainty instead of hiding it.

Negation

Denied and ruled-out diagnoses are flagged and not accepted.

Specificity

General findings become documentation needs when detail is missing.

Validation

Codes absent from the configured corpus are rejected as `not_in_configured_corpus`.

Demo Surface
